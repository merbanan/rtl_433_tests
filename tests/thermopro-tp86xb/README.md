# ThermoPro TempSpike XR TP862b / TP863b Wireless Dual-Probe Meat Thermometer.

[TP862b product page](https://temppro.com/products/tp862b-tempspike-xr-wireless-meat-thermometer)
[TP863b product page](https://temppro.com/products/tempspikexr-tp863b-2000ft-wireless-meat-thermometer)

Both thermomenters share the same probes, and a base station (albeit with a different FCC ID). The only difference is that TP863b comes with a separate booster unit which amplifies and re-transmits the probe signals.
