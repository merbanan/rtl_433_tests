# Novo blinds remote (868 MHz)

## Manufacturer

- Novo, https://en.gznovo.com/

Sold in Canada by Maxxmar as remote for skylight blinds.

## Models

- Novo D33100

## Button operation

The top three buttons control blinds movement: up, stop and down. The channel used is shown on the remote's "screen" when a button is pressed.

The bottom button `<>` is used to iterate through channel 1 to 6 or select all of them.. not sure purpose of that is.

The battery compartment has P1 and P2 buttons which are used to program blinds movement.
