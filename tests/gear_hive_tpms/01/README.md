# Gear-Hive unbranded aftermarket TPMS

Unbranded aftermarket TPMS sensor using CMT2220LY receiver module at 433.92 MHz.
OOK modulation with Manchester encoding (MC_ZEROBIT), 186-bit packets.

Sensors recorded: Rear Left (ID 2b4ea1) and Front Right (ID 29b6a2).
g002 also contains a third burst with Rear Left at higher pressure (counter 3949, 218.8 kPa).
Raw captures from a vehicle with 4 sensors; only the left side was close enough to record.
