# Inkbird IBS-P04R

A 433 MHz FSK wireless pool water quality sensor (temperature + TDS).

https://www.ink-bird.com/

## Notes

Samples captured from a single IBS-P04R sensor at 433.92 MHz, 250k sample rate.
Temperature values range from 11.3-18.1 C, TDS from 192-199 ppm.
All samples verified with CRC-16/MODBUS.
