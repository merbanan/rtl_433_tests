# Watts WFHT-RF / WFHC-MASTERH&C-RF

Wireless underfloor heating thermostat.

Frequency: 433.92 MHz

Modulation:

- OOK
- Manchester encoded

Sample files

## turning_thermostat_off_433.92M_250k.cu8

Turning the thermostat off results in a setpoint of 0.0 °C.

## turning_thermostat_on_433.92M_250k.cu8

Turning the thermostat on results in the configured setpoint being transmitted.

## pairing_thermostat_433.92M_250k.cu8

Captured while the thermostat is in pairing mode.
