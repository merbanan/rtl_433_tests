# OWL CM130 energy monitor
#
# Source: https://github.com/merbanan/rtl_433/issues/1493
#
# Oregon-Scientific v3 family, OOK Manchester, 97-bit message, preamble
# 00 00 00 60. Codes below are the message as printed by the prototype
# feat-cm130 decoder ("orig" form, MSB nibble order); byte 0 reads 0x00
# because the prototype masks the 0x60 sync nibble (real wire byte is 0x60).
#
# No CM130 decoder is merged yet, so these are documentation only (there is
# no codes_test.json). Known field decoding, worked out from this data:
#
#   after reflecting nibbles of the message:
#   - id        = byte 2
#   - power_W   = ((b[3] << 8) | (b[5] >> 4)) / 16      (confirmed vs display)
#   - energy    = 36-bit LE counter in bytes 6..9 (+ low nibble of b5),
#                 ~256 J per count; absolute kWh offset still unknown
#   - byte 11   = checksum, provably linear over GF(2) but formula unsolved
#
# See ../readme.txt for the full status. The reset-button captures in 01/
# carry no meaningful reading and are documented in 01/readme.txt.
