# Baldr E0666TH Thermo-Hygrometer

Captures taken from three different sensors which were purchased together as a
set with a base station. Captures were recorded with `-Y autolevel` and require
passing `-Y minlevel=-27` to decode successfully. All values in the JSON files
(except the device ID) were verified against the display on the base station.

The sensor transmits at random intervals, approximately every two minutes.
